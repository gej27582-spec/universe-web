# 深空观测站 / Deep Space Observatory

一个以动态太阳系为主体的桌面网页交互原型。v0.2 加入目标镜头飞行、自动扫描、观测详情、地球比例对比、木星四大卫星与自动巡航。

## 运行

```powershell
npm install
npm run dev
```

浏览器打开 `http://127.0.0.1:4173/`。

## 操作

- 在太阳系区域拖动鼠标：旋转观察角度。
- 使用滚轮：拉近或拉远视角。
- 点击太阳或任意行星：执行镜头飞行与扫描锁定，并查看概览、参数和对比信息。
- 使用右侧天体索引：快速选择目标。
- 聚焦木星后：点击卫星影像或左侧次级目标列表查看四大卫星信息。
- 使用底部控制：暂停轨道、切换时间速度或启动自动巡航。
- 自动巡航中进行拖拽、缩放或手动选择会立即退出巡航。
