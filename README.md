# 深空观测站 / Deep Space Observatory

一个以动态太阳系为主体的桌面网页交互原型。

## 运行

```powershell
npm install
npm run dev
```

浏览器打开 `http://127.0.0.1:4173/`。

## 操作

- 在太阳系区域拖动鼠标：旋转观察角度。
- 使用滚轮：拉近或拉远视角。
- 点击太阳或任意行星：聚焦天体并查看信息。
- 使用右侧天体索引：快速选择目标。
- 使用底部控制：暂停轨道或切换时间速度。

## 检查

```powershell
npm run typecheck
npm test
npm run build
```
