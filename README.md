# 深空观测站 / Deep Space Observatory

一个以动态太阳系为主体的桌面网页交互原型。v0.3「卫星世界」加入十颗可聚焦卫星、局部真实比例、专属天体现象与艺术化环境音景；v0.3.4 重点加固中文可读性、观测引导、资源审计和维护性。

## 运行

```powershell
npm install
npm run dev
```

浏览器打开 `http://127.0.0.1:4173/`。

## 操作

- 在太阳系区域拖动鼠标：旋转观察角度。
- 使用滚轮：拉近或拉远视角。
- 点击太阳或任意行星：执行镜头飞行与扫描锁定，并查看概览、参数和对比信息。
- 使用右侧天体索引：快速选择目标。
- 聚焦地球、火星、木星、土星或海王星后：从右侧索引、左侧卫星列表或场景中选择卫星。
- 卫星目标会执行独立镜头飞行、扫描和锁定，可查看概览、现象与参数。
- 使用“展示／真实”切换局部卫星系统比例；太阳系全景始终使用艺术化压缩比例。
- 点击右上角 SOUND 开启授权艺术化音景；声音并非真实太空录音。
- 使用底部控制：暂停轨道、切换时间速度或启动自动巡航。
- 自动巡航中进行拖拽、缩放或手动选择会立即退出巡航。
