# v0.4 太阳系边缘素材与资料来源

本项目新增天体遵循“真实来源优先，找不到可靠贴图就明确标注 fallback”的规则。禁止使用来源不明、授权不清或随机生成的贴图伪装真实素材。

## 新增天体

| 天体 | 可视化方式 | 来源 | 授权 / 使用说明 | 用途 |
| --- | --- | --- | --- | --- |
| 冥王星 | 真实全球拼接图 | USGS Astrogeology / NASA New Horizons Pluto Global Mosaic | 页面标注 Access Constraints: None；Use Constraints: cite authors | `public/textures/outer-system/pluto-usgs-new-horizons-1024.jpg` |
| 卡戎 | 真实全球拼接图 | USGS Astrogeology / NASA New Horizons Charon Global Mosaic | 页面标注 Access Constraints: None；Use Constraints: cite authors | `public/textures/outer-system/charon-usgs-new-horizons-1024.jpg` |
| 谷神星 | 程序 fallback | NASA Ceres Facts / Dawn 任务资料 | 当前未接入可直接网页分发的全球贴图；详情页明确标注为科学资料约束的可视化 | 灰暗冰岩表面、陨石坑、Occator 亮斑提示 |

## 当前缺口

- 谷神星、小行星带代表体、彗星代表体尚未接入真实全球贴图。
- 在找到可靠可分发素材前，不应替换为随机网络图。
- 后续如果接入新贴图，必须同步更新 `src/lib/solarSystem.ts` 的 `asset` 字段和本文件。
