全局调整：

- 主行星贴图加载失败时使用有身份特征的程序 fallback，不再是纯色球。
- 彩色行星贴图加载后统一经过 `createPlanetDisplayTexture` 校准，降低过曝、纯色和塑料感。
- 聚焦行星/卫星时降低行星轨道线和卫星轨道线透明度，减少辅助线横穿天体正面的视觉干扰。
- 画质输出固定为最高画质；卫星系统、音频降级和 GitHub Pages 流程不变。
