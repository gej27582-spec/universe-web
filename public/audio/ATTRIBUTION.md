# Observatory soundscape

The ambient loop and interface cues come from the **50 CC0 Sci-Fi SFX** pack
published on OpenGameArt under the Creative Commons Zero (CC0) dedication.

- `ambient-loop.ogg`: source file `loop_ambient_01.ogg`
- `flyby.ogg`: source file `teleport_01.ogg`
- `scan-lock.ogg`: source file `terminal_03.ogg`

Source: https://opengameart.org/content/50-cc0-sci-fi-sfx

The soundscape is an artistic interface treatment. It is not presented as
recorded sound propagating through space.
