# Solar System Scope textures

The texture files in this directory are sourced from
[Solar System Scope](https://www.solarsystemscope.com/textures/).

They are distributed under the
[Creative Commons Attribution 4.0 International license](https://creativecommons.org/licenses/by/4.0/).

Changes made for this project: the original files are served locally and used
with real-time lighting, color management, atmosphere layers, and geometry in
Three.js. No source image pixels were altered.
